### git command
创建分支
```
git branch name  
```
切换分支位置
```
git checkout name
```
查看分支及位置
```
git branch 
```
合并分支，合并前要切换到主分支，name是被合并的小的分支
```
git branch name  
```
主分支上传
```
git push 
```
小分支上传
```
git push -u main login
```
